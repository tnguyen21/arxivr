About

arxivr makes browsing and finding papers on arXiv easier.

Inspired by arxiv-sanity-lite.com

How it works

arxivr uses a combination of common search techniques and vector search to find papers.

Abstracts have been pulled from the following categories: cs.CL, cs.AI, cs.MA, cs.CV, cs.LG, cs.RO, cs.SY, cs.SI, cs.HC, cs.IR, between the years 2021-2025. They're stored in a local sqlite database, which is queried on every page load.

The vector search is powered by siglip, a vision-language model, and hnswlib for approximate nearest neighbor search.
